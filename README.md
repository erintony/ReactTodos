# 项目说明
使用react.js ,模仿Backone.js todos 实现

## 环境
- react 
- Es2015
- babel
- webpack

## 下载依赖模块
- 在项目目录下运行 `npm install`
## 运行
- 方式一
1. 在项目目录运行`webpack`
2. 在浏览器打开index.html

- 方式二 监听更新，实时编译
1. 在项目目录运行 `npm run build`
